# WordPress Custom Theme Project

## 📌 Overview

This project contains a custom WordPress theme built using ACF Flexible Content.

## ⚙️ Requirements

* WordPress (latest)
* PHP 7.4+
* MySQL
* Advanced Custom Fields (ACF) PRO plugin

## 🚀 Setup Instructions

1. Download WordPress from https://wordpress.org

2. Copy `wp-content` folder from this repo into your WordPress installation

3. Install and activate plugin:

   * Advanced Custom Fields PRO
   * Gravity Forms
   * Yoast SEO

4. Activate the theme:
   Appearance → Themes → dagmar

5. ACF fields will automatically load from `acf-json`

6. Create pages manually and assign templates

## 🧩 Notes

* No database is included (for security reasons)
* Media/uploads are not included
* This repo contains only theme and configuration
